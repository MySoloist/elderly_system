# 老年人社区餐饮配送系统 - 管理端UI设计文档

## 1. 设计概述

### 1.1 设计理念

本设计以管理员的工作需求为核心，遵循"专业、高效、直观"的设计原则，通过现代化的UI设计、清晰的信息层次和便捷的操作流程，确保管理员能够高效地完成各项管理任务。

### 1.2 目标用户

- 年龄：25-50岁的系统管理员
- 特点：熟悉计算机操作，注重工作效率，需要处理大量信息
- 需求：专业的管理界面、清晰的数据展示、便捷的操作流程

### 3.2 侧边栏设计

**整体布局**：固定宽度280px的侧边导航栏

**实际实现细节**：

- **Logo区域**（高度自适应）：
  - 图标（Food图标，32x32px，白色），渐变背景，带有浮动动画效果
  - 文字：颐养膳食（20px，白色，粗体，字间距2px）
  - 副标题：社区老人餐管理系统（11px，浅灰色，大写，字间距1px）
  
- **导航菜单**：
  - 菜单项分组显示（核心管理、运营调度、服务质量）
  - 分组标题（11px，浅灰色，大写，间距8px，字间距1px）
  - 菜单项：图标（20x20px） + 文字（14px）
  - 当前选中菜单项高亮显示（渐变背景，文字白色，平移8px效果）
  - 菜单项背景带有透明边框和渐变效果
  
- **交互效果**：
  - 菜单项悬停时背景色变深，轻微平移（4px）
  - 选中菜单项有动画效果（平移8px）
  - 图标和文字有过渡动画
  - 支持深色/浅色主题切换

### 3.3 顶部导航栏设计

**整体布局**：水平导航栏，包含面包屑、主题切换、通知、用户信息

**实际实现细节**：

- **左侧区域**：
  - 面包屑导航（当前位置：首页 > 工作台概览）
  - 面包屑带有渐变背景和毛玻璃效果
  
- **右侧区域**：
  - 主题切换按钮（月亮/太阳图标，24x24px），支持深色/浅色主题切换
  - 通知按钮（铃铛图标，带消息数量徽章）：
    - 点击显示通知下拉菜单（350px宽度）
    - 通知列表显示配送异常通知
    - 支持标记全部已读功能
  - 用户信息下拉菜单：
    - 用户头像（40x40px，圆形）
    - 用户姓名（15px，粗体）
    - 用户角色（12px，灰色）
    - 下拉选项：个人中心、系统设置、退出登录
    - 用户信息卡片带有毛玻璃效果

### 3.4 主内容区域设计

**整体布局**：自适应的内容区域，根据不同页面显示不同内容

- **页面标题区域**：
  - 大标题（28px，粗体）
  - 副标题（14px，灰色）
  - 操作按钮（右侧）
  
- **内容区域**：
  - 根据页面类型显示不同的内容（表格、卡片、图表等）
  - 支持响应式布局，适配不同屏幕尺寸
  
- **动画效果**：
  - 页面切换时的淡入淡出动画（300ms）
  - 内容加载时的骨架屏效果
  - 数据更新时的过渡动画

## 5. 交互设计

### 5.1 按钮交互

- **点击反馈**：按钮点击时颜色加深，轻微缩放（0.98倍）
- **悬停效果**：按钮背景色变浅，添加阴影效果
- **禁用状态**：按钮灰色，不可点击，文字灰色
- **加载状态**：显示加载动画，按钮不可点击

### 5.2 表格交互

- **行悬停**：背景色变浅，光标变为指针
- **行选中**：背景色加深，显示选中状态
- **排序功能**：点击表头可排序，显示排序图标
- **分页切换**：平滑过渡，保持当前选中状态

### 5.3 表单交互

- **输入反馈**：输入框获得焦点时边框变色（主色调）
- **实时验证**：输入时实时验证，显示成功/错误图标
- **错误提示**：输入错误时显示红色文字提示
- **表单提交**：提交时显示加载状态，提交成功后显示成功提示

### 5.4 特殊交互

- **主题切换**：平滑过渡动画，保存用户偏好
- **抽屉滑出**：从右侧滑出，带半透明背景遮罩
- **弹窗显示**：居中显示，带背景遮罩和动画效果
- **通知提醒**：右上角弹出，自动消失
- **表格主题切换**：深色主题下表格样式自动调整（背景色#1a1a2e，交替行背景色）

## 6. 特殊设计考虑

### 6.1 深色主题支持

- **主题切换**：支持深色/浅色主题切换
- **颜色适配**：所有颜色都有对应的深色主题版本
- **表格样式**：深色主题下表格样式自动调整
- **保存偏好**：记住用户的主题偏好设置（localStorage）

### 6.2 响应式设计

- **屏幕适配**：支持不同屏幕尺寸（1200px以上最佳）
- **布局调整**：小屏幕下自动调整布局
- **移动端支持**：关键功能支持触摸操作

### 6.3 性能优化

- **懒加载**：图片和组件懒加载
- **虚拟滚动**：大量数据表格使用虚拟滚动
- **缓存机制**：缓存常用数据，减少请求
