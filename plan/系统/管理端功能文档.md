# 老年人社区餐饮配送系统 - 管理端功能文档

## 1. 项目概述

本项目是一个基于 Vue 3 + Element Plus 开发的老年人社区餐饮配送管理系统，提供完整的后台管理功能，包括老人管理、订单处理、配送调度、餐品管理等核心功能。

## 2. 技术架构

### 2.1 前端技术栈

- **框架**: Vue 3 (Composition API)
- **UI组件**: Element Plus
- **路由**: Vue Router 4
- **图标库**: Element Plus Icons
- **构建工具**: Vite
- **图表库**: 自定义CSS实现柱状图趋势展示
- **地图服务**: 百度地图API

### 2.2 界面特点

- **现代化设计**: 采用现代化的UI设计，支持深色/浅色主题切换
- **响应式布局**: 适配不同屏幕尺寸
- **流畅动画**: 页面切换和交互都有平滑的动画效果
- **毛玻璃效果**: 多处使用glass-effect视觉设计

**功能描述**: 系统的总览页面，展示关键业务数据和待办事项

**文件位置**: `src/views/Dashboard.vue`

**主要功能**:

  - 展示最近7天订单数量变化
  - 自定义CSS柱状图，带悬停效果和提示框

- **每日收益趋势图**（右侧大卡片）:
