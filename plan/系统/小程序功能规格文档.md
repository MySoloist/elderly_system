# 老年人社区餐饮配送系统 - 三端小程序功能规格文档

## 1. 项目概述

本项目使用 uniapp 框架开发三个小程序端，分别为老人端、家属端和配送员端，旨在为老年人提供便捷的餐饮配送服务，同时让家属能够放心地为老人订餐，让配送员高效地完成配送任务。

## 2. 老人端小程序功能规格

### 2.1 核心功能

#### 2.1.1 用户登录与注册

- **账号密码登录**：统一使用账号密码登录方式
- **微信一键登录**：支持微信授权快速登录
- **账号注册**：支持新用户注册账号
- **密码找回**：标记为"后续开发中"状态
- **登录动画效果**：温馨的老人形象和食物图标动画，背景食物图标悬浮效果
- **个人信息完善**：设置紧急联系人、健康状况等信息

#### 2.1.2 悬浮AI智能助手

- **悬浮设计**：AI助手以悬浮图标的形式显示在界面右下角，可随时点击调用
- **可拖拽窗口**：支持拖拽调整位置，方便灵活使用
  - 拖拽顶部栏即可移动窗口位置
  - 自动限制在屏幕范围内
- **最小化功能**：支持窗口最小化，节省屏幕空间
  - 最小化后显示为 compact 状态栏
  - 点击可恢复完整窗口
- **智能餐品推荐**：根据老人的口味偏好、健康状况、饮食禁忌等智能推荐适合的餐品
- **语音交互**：支持语音对话方式与AI助手交流
  - 集成阿里云语音识别（ST）服务
  - 实时语音识别结果显示
  - 录音权限管理和错误处理
- **语音合成**：支持AI回复语音播报
  - 集成阿里云语音合成（TTS）服务
  - 使用小云（标准女声）语音
  - 支持音量、语速、音调调节
- **打字聊天**：支持文字输入方式与AI助手交流对话
- **健康建议**：基于老人的健康数据提供饮食建议和健康提示
- **操作提醒**：关键操作步骤的语音提醒和确认
- **随时调用**：老人可以在任何页面随时点击悬浮图标召唤AI助手
- **对话历史**：加载历史对话记录，保持上下文连续性
- **语音播放按钮**：每条AI消息旁显示播放按钮，支持重复播放

#### 2.1.3 餐品浏览与下单

- **简化界面**：大字体、清晰图标，适合老年人使用
- **搜索功能**：支持按餐品名称或标签搜索
- **AI智能推荐**：根据健康状况和饮食偏好智能推荐餐品
  - AI思考状态显示：显示"AI正在分析您的健康数据和饮食偏好..."
  - 推荐轮播展示：使用swiper组件横向滚动展示推荐餐品
  - 推荐理由展示：显示每个推荐餐品的推荐理由标签
- **健康数据展示**：可展开查看个人健康数据（身高、体重、血压、血糖、饮食偏好、过敏史、用药情况、医生建议）
- **常用餐品收藏**：快速选择常点餐品，支持一键收藏/取消收藏
- **餐品分类**：营养套餐、时令蔬菜、精选肉类、流食软食等分类（动态从API获取标签）
- **餐品详情**：包含图片、价格、营养成分、适合人群等信息
- **快速点餐**：支持一键点餐功能
  - 即时下单：立即创建订单
  - 预定下单：支持选择日期和时间进行预定
- **时间选择器**：自定义日期时间选择器，支持选择配送日期、小时（8-20点）、分钟

#### 2.1.4 订单管理

- **我的订单**：查看历史订单和当前订单状态
- **订单状态跟踪**：实时查看订单进度
  - 待支付：订单等待支付
  - 待接单：订单等待配送员接单
  - 配送中：配送员正在配送
  - 已完成：订单已完成
  - 已取消：订单已取消
  - 已预定：预定订单
- **订单筛选**：支持按订单状态筛选（全部、待支付、待接单、配送中、已完成、已取消、已预定）
- **订单类型标识**：区分即时订单和预定订单，显示预定时间
- **订单操作**：
  - 取消订单：取消待支付或等待预定时间的订单
  - 去支付：支付待支付订单
  - 评价：对已完成的订单进行评价（支持餐品评价和配送员评价）
  - 联系骑手：支持语音通话和短信联系配送员
  - 查看详情：查看订单详细信息
- **订单评价功能**：
  - 评价类型选择：餐品评价或配送员评价
  - 星级评分：5星评分系统
  - 快捷评价标签：非常满意、味道很好、配送及时、包装完好、营养均衡、服务周到
  - 评价内容输入：支持多行文本输入，最多200字
  - 图片上传：支持上传最多9张图片
  - 配送员信息展示：显示配送员姓名和电话

#### 2.1.5 个人中心

- **个人信息管理**：修改个人资料、上传头像
- **饮食偏好设置**：设置饮食禁忌、口味偏好
- **健康档案**：记录健康状况、疾病史、用药情况等
- **我的收藏**：查看和管理收藏的餐品
- **我的评价**：查看历史评价记录
- **我的公告**：查看系统公告和通知
  - 分类筛选：全部、系统公告、通知、紧急通知
  - 未读标记：未读公告显示红点提示
  - 阅读状态：本地存储已读公告ID
- **健康提醒**：查看健康提醒记录
- **紧急联系人**：
  - 自动同步：自动从绑定的家属中获取紧急联系人
  - 主要联系人设置：支持设置主要紧急联系人
  - 联系方式：电话、短信、消息多种联系方式
  - 快捷消息：支持发送预设消息（我饿了、我不舒服、我想你了）或自定义内容
  - 紧急求助：支持拨打110、119、120等紧急电话
- **设置**：账号设置和系统设置（后续开发中）
- **退出登录**：支持退出当前账号

### 2.2 用户体验优化

#### 2.2.1 界面设计

- **大字体模式**：支持调整字体大小
- **高对比度模式**：提高文字与背景的对比度
- **语音提示**：关键操作提供语音提示
- **简化操作流程**：减少操作步骤，关键功能一键可达

#### 2.2.2 交互优化

- **语音辅助**：支持语音控制和语音反馈
- **手势操作**：支持简单的手势操作
- **容错处理**：误操作提醒和确认机制

## 3. 家属端小程序功能规格

### 3.1 核心功能

#### 3.1.1 用户登录与绑定

- **账号密码登录**：统一使用账号密码登录方式
- **微信一键登录**：支持微信授权快速登录
- **账号注册**：直接注册账号并设置密码
- **密码找回**：通过账号找回密码
- **登录页面设计**：
  - 背景装饰：3个浮动圆形（大/中/小），带模糊效果和浮动动画
  - Logo：80x80px，渐变背景（#6366f1 → #8b5cf6），带发光动画
  - 登录卡片：圆角32px，毛玻璃效果（backdrop-filter: blur(20px)），带渐变边框
  - 输入框：56px高度，16px字体，聚焦时边框变色和阴影效果
- **老人绑定**：绑定多位老人账号，统一管理
  - 支持搜索和选择老人
  - 显示老人状态（正常、注意、关注）
- **权限设置**：设置对老人账号的操作权限

#### 3.1.2 老人管理

**实际实现细节**：
- **老人列表**：查看绑定的所有老人信息，显示老人状态（正常、注意、关注）
  - 在个人中心页面以可展开列表形式展示
  - 支持快捷操作：编辑口味、编辑健康
- **老人档案**：查看和编辑老人的详细信息
- **健康记录**：记录和编辑老人的健康状况和饮食情况
- **饮食健康查看**（首页/健康页）：
  - 老人选择器：下拉选择绑定的老人，显示老人状态标签
  - 营养摄入卡片：显示当日热量、蛋白质、脂肪、碳水化合物摄入量
    - 主热量显示：24px粗体，渐变文字效果
    - 详细营养：蛋白质、脂肪、碳水三列显示
  - 饮食规律卡片：显示早中晚三餐状态和时间
    - 圆形状态指示器：已完成显示渐变背景，未完成显示灰色背景
  - 健康状态卡片：显示血压、血糖等健康数据
    - 健康标签：彩色标签显示（低糖、低盐等）
  - 日期选择器：支持前后切换日期查看历史记录
  - 饮食记录列表：按时间段显示餐次，包含餐品名称、热量、营养标签
    - 卡片式布局，圆角24px，阴影效果
    - 显示餐品图片、名称、热量
    - 营养标签集合显示
  - 健康趋势图表：营养摄入趋势可视化（开发中）
  - AI智能健康建议：基于老人健康数据提供个性化饮食建议
    - 渐变背景卡片（#6366f1 → #8b5cf6 → #ec4899）
    - 带发光脉冲动画效果
- **口味偏好编辑**：编辑老人的饮食口味偏好和禁忌
- **健康情况编辑**：编辑老人的健康状况、疾病史、用药情况等
- **紧急联系人**：设置和管理老人的其它紧急联系人
- **健康提醒功能**：
  - 支持发送饮食提醒、健康建议、体检提醒
  - 可自定义提醒内容和时间
  - 弹窗形式：选择提醒类型、输入内容、设置时间
  - 时间格式：YYYY-MM-DD HH:MM

#### 3.1.3 代下单功能

**实际实现细节**：
- **为老人点餐**：查看老人的餐品推荐并下单，支持按分类筛选（主食、荤菜、素菜、汤品、点心）
- **批量订餐**：支持选择多位老人同时下单，老人选择器显示老人健康标签（低糖、低盐、清淡、低脂、软食）
- **搜索功能**：支持按餐品名称或标签搜索
- **餐品管理**：
  - 餐品卡片网格布局（2列），间距24px
  - 餐品卡片显示名称、价格、营养标签（碳水、蛋白质、维生素、膳食纤维等）
  - 餐品图片：支持显示和错误处理（加载失败显示占位图）
  - 收藏功能：支持收藏/取消收藏餐品
  - 支持数量调整和加入购物车功能
    - 数量控制：-/+按钮，实时显示数量
    - 未选择时显示+按钮
  - 购物车弹窗显示已选餐品列表和总价
    - 从底部滑出
    - 支持调整数量
    - 显示总价和确认订单按钮
- **订单类型选择**：
  - 现在下单：立即创建订单
  - 预定下单：选择日期和时间进行预定
  - 时间选择器：日期选择 + 小时（8-20点）+ 分钟选择
- **订单提交**：确认订单后清空购物车并跳转至订单管理页面
- **订单支付**：支持微信支付、支付宝等多种支付方式（当前为模拟实现）

#### 3.1.4 配送跟踪

- **实时配送状态**：查看订单的实时配送状态
- **配送员信息**：查看配送员的姓名、联系方式
- **预计送达时间**：显示预计送达时间
- **异常处理**：处理配送过程中的异常情况

#### 3.1.5 消费管理

- **消费记录**：查看老人的消费历史
- **消费统计**：按时间段统计消费金额和餐品偏好
  - 时间范围选择：今日、本周、本月、自定义
  - 统计卡片：总消费金额、订单数量、平均每单金额
  - 消费趋势图表：柱状图/折线图展示
- **账单明细**：查看详细的消费账单
  - 按日期分组显示
  - 每笔记录显示：餐品名称、数量、金额、时间
- **优惠活动**：查看和使用优惠活动

### 3.2 通知与提醒

#### 3.2.1 配送通知

- **订单状态通知**：订单状态变化时同时通知家属和老人
- **配送异常提醒**：配送延迟或异常时同时提醒家属和老人
- **送达确认**：餐品送达后同时通知家属和老人

#### 3.2.2 健康提醒

- **饮食提醒**：提醒老人按时用餐
- **健康建议**：基于老人健康状况提供饮食建议
- **体检提醒**：定期提醒老人进行健康检查

## 4. 配送员端小程序功能规格

### 4.1 核心功能

#### 4.1.1 用户登录与认证

- **账号密码登录**：统一使用账号密码登录方式
- **微信一键登录**：支持微信授权快速登录
- **账号注册**：直接注册账号并设置密码
- **密码找回**：通过账号找回密码
- **登录页面设计**：
  - Logo：配送员图标🚴‍♂️，渐变背景，带发光动画
  - 配送动画：云朵装饰、餐品图标、电动车动画
  - 登录表单：圆角卡片，阴影效果
  - 输入框：56px高度，带图标，密码显示切换
  - 微信登录按钮：绿色背景，带微信图标

#### 4.1.2 订单管理

**实际实现细节**：
- **订单列表**：
  - 支持按状态筛选（待接单、配送中、已完成）
  - 订单卡片显示：
    - 订单编号：14px灰色文字
    - 订单状态标签：彩色标签（待接单-橙色、配送中-蓝色、已完成-绿色）
    - 管理员委派标签：显示"该订单已被管理员委派"
    - 老人信息：头像👴 + 姓名 + 脱敏电话
    - 社区信息：🏘️图标 + 社区名称
    - 配送地址：📍图标 + 详细地址
    - 餐品信息：餐品名称 + 总价 + 数量
    - 预计送达时间和距离
  - 支持点击查看订单详情
- **订单详情**：查看订单的详细信息，包括老人信息、配送地址、餐品信息等
- **接单操作**：
  - 待接单状态：支持一键接单，接单后订单状态变为配送中
    - 判断条件：未被委派或委派给当前配送员
  - 配送中状态：支持确认送达，送达后订单状态变为已完成
  - 已完成状态：支持联系老人/家属功能
- **订单状态更新**：更新订单状态（待接单→配送中→已完成）
- **联系老人/家属**：支持直接拨打老人或家属电话（电话号码部分隐藏，拨打时自动替换）
- **距离计算**：
  - 优先使用后端提供的老人位置信息计算距离
  - 支持地址解析计算实际距离
  - 实时更新距离显示

#### 4.1.3 配送导航

**实际实现细节**：
- **地图导航**：
  - 使用uni-app原生map组件
  - 显示当前位置到目的地的导航路线
  - 显示配送员位置标记和老人位置标记
  - 支持缩放和滚动
- **专注模式**：
  - 支持切换专注模式，专注模式下地图占满整个屏幕
  - 提升导航体验，减少干扰
  - 按钮显示"专注模式"/"退出专注"
- **多订单管理**：
  - 支持同时管理多个配送订单
  - 订单切换组件：显示订单序号、老人姓名、状态标签
  - 管理员委派标签显示
  - 完成当前订单后自动切换到下一个未完成订单
- **配送状态监控**（状态网格）：
  - 配送状态卡片：🚚图标 + "配送中"/"待配送"
  - 预计送达卡片：⏰图标 + 预计时间
  - 距离卡片：📍图标 + 距离信息
- **配送信息卡片**：
  - 老人头像👴 + 姓名 + 地址 + 餐品名称
- **快捷操作按钮**：
  - 开始导航：🧭图标
  - 异常处理：⚠️图标，警告色
  - 已送达：✅图标，主色调
- **计时功能**：
  - 自动计时显示已配送时间
  - 页面卸载时清除计时器

#### 4.1.4 异常处理

- **配送延迟**：因特殊情况无法按时配送时提交延迟申请
- **配送异常**：处理配送过程中的异常情况（如老人不在家、地址错误等）
- **联系客服**：遇到问题时联系客服寻求帮助

#### 4.1.5 个人中心

- **个人信息管理**：
  - 修改个人资料、上传头像
  - 显示用户姓名、脱敏手机号、状态
  - 头像支持点击更换
- **配送统计**（4列网格）：
  - 今日订单：数字显示
  - 总订单：累计订单数量
  - 好评率：百分比显示
  - 总收入：金额显示
- **功能菜单**：
  - 个人信息 👤
  - 我的评价 ⭐
  - 消息中心 📩（带未读消息徽章）
  - 排班管理 📅
