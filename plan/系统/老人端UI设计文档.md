# 老年人社区餐饮配送系统 - 老人端UI设计文档

## 1. 设计概述

### 1.1 设计理念

本设计以老年人的使用需求为核心，遵循"简洁、清晰、易用"的设计原则，通过大字体、高对比度、充足间距和直观交互，确保老年人能够轻松使用小程序。

### 1.2 目标用户

- 年龄：60岁以上老年人
- 特点：视力可能下降、手指灵活性降低、学习能力减弱
- 需求：简单直观的操作界面、清晰的视觉反馈、便捷的交互方式

## 3. 页面设计详情

### 3.1 登录注册页面（非TabBar页面）

**触发条件**：用户首次打开小程序或退出登录后

**整体布局**：居中布局，温馨动画效果

- **主题动画元素**：
  - 老奶奶形象（👵）：48px，elderlyWave动画（3s ease-in-out infinite）
  - 老爷爷形象（👴）：48px，elderlyWave动画（3s ease-in-out infinite）
  - 餐盘（🍱）：40px，plateRotate动画（4s linear infinite）
  - 周围食物图标：米饭🍚、蔬菜🥗、汤🍲、饺子🥟、面条🍜、鸡蛋🥚、肉类🥩，foodFloat动画（3s ease-in-out infinite，不同延迟）
- **背景装饰动画**：
  - 8个食物图标（🍞🥖🥟🍤🍎🍊🍌🍋）分散在页面四周
  - 24px大小，opacity: 0.08
  - bgFloatAnimation动画（8s ease-in-out infinite，不同延迟）
- **顶部区域**（占页面30%）：
  - 系统Logo（颐养膳食）- 50x50px图标 + 28px文字
  - 副标题：老人订餐系统 - 28px文字
- **中间区域**（占页面40%）：
  - 登录表单卡片（白色背景，圆角20px，阴影效果，backdrop-filter: blur(20px)）
  - 账号输入框：56px高度，左侧👤图标，18px字体，圆角12px
  - 密码输入框：56px高度，左侧🔒图标，右侧眼睛图标（👁️/🙈），18px字体
  - 登录按钮：56px高度，渐变主色调背景（#FF7A45 → #FF8F6A），白色文字，圆角12px，阴影效果
  - 微信一键登录按钮：56px高度，渐变主色调背景，💬图标 + "微信一键登录"文字
- **底部区域**（占页面30%）：
  - 注册新账号链接：18px文字，#409EFF蓝色
  - 找回密码链接：18px文字，#409EFF蓝色
  - 版权信息：14px文字，灰色

**设计特点**：

- 温馨的动画效果，提升用户体验
- 老人挥手动画和食物悬浮动画营造温暖氛围
- 表单居中显示，减少视觉干扰
- 输入框和按钮尺寸足够大，方便点击
- 文字对比度高，确保清晰可读
- 背景渐变效果（#FFF7F3 → #FFFAF7），营造温暖氛围
- 支持微信一键登录，简化登录流程

**动画定义**：
```css
@keyframes elderlyWave { 0%,100% { transform: rotate(0deg); } 25% { transform: rotate(-5deg); } 75% { transform: rotate(5deg); } }
@keyframes plateRotate { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } }
@keyframes foodFloat { 0%,100% { transform: translateY(0px); } 50% { transform: translateY(-10px); } }
@keyframes bgFloatAnimation { 0%,100% { transform: translateY(0px) rotate(0deg); } 50% { transform: translateY(-30px) rotate(15deg); } }
```
